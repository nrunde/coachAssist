CoachAssist

CoachAssist ist ein kleines Tool für Amateur Fußballtrainer um den Kader für ein Spiel schnell, einfach und übersichtlich verwalten zu können.

- Spielerliste pro Spiel
- Spieler mit einem Klick entfernen
- Spieler per Drag&Drop oder mit einem Klick in die Startelf wählen
- mehrere Spiele verwalten
- automatisches Speichern bei Änderungen

Der CoachAssist wird bald online in einer Beta Version erreichbar sein.
In Zukunft soll der CoachAssist auch auf dem Smartphone vernünftig funktionieren.

In späteren Versionen sind noch Funktionen angedacht wie 

- Saison Kaderverwaltung (erweiterte Spielerinformationen wie z.B. Kontaktdaten, Verletzungen etc.)
- Synchronisierung zwischen verschiedenen Geräten
- mobile Apps
- Live-Spiel Funktionen (Tore, Ein-/Auswechslungen, Karten etc.)
- usw. 

Feature Requests bitte in den Issue Tracker.
